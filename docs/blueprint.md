# **App Name**: K-Culture Compass

## Core Features:

- K-Pop News Feed: Aggregates K-Pop news and trends from various sources.
- K-Beauty Trends: Displays the latest K-Beauty trends, tips, and product reviews.
- Interactive Map: Full-screen map with customizable layers for discovering K-Culture hotspots.
- Location Filtering: Filters locations on the map based on categories like Pharmacy, K-Pop Holy Lands, K-Pop Haul, Food/Cafe, and Dermatology. Display pharmacies that shows the prices of cosmetic items.
- Informative Posts: Connects map locations to detailed informational posts (e.g., treatments offered at dermatology clinics).
- Multi-language Support: Supports English, Chinese, and Japanese languages with easy toggling. LLM will act as a tool, to make sure the translations of informal information (e.g. blog posts) preserve their connotations and style.
- User Reviews: Allows users to post their reviews for the listed locations. LLM acts as a tool to suggest better review wording, summarization of multiple reviews.

## Style Guidelines:

- Primary color: Vibrant pink (#F06292) to represent the energy of K-Pop and K-Beauty.
- Background color: Light pink (#FCE4EC), a very desaturated version of the primary, providing a soft backdrop.
- Accent color: Purple (#BA68C8), for interactive elements and highlights, providing a vivid contrast.
- Headline font: 'Poppins' (sans-serif) for a contemporary, precise feel. 
- Body font: 'PT Sans' (sans-serif), a pairing choice for 'Poppins' with enhanced readability.
- Clean and modern icons representing each category (pharmacy, K-Pop landmarks, etc.).
- Mobile-first, responsive design with clear navigation between the news feed and the interactive map.
- Subtle animations for transitions and loading states to enhance user experience.
